Detailed Description

Manages a list of decoder and reader instances using an ID system.

This class implements the C API version of the decoder library. It allows for the simultaneous operation of one or more decoder and reader instances.

For more information, see the section for Using the C API.

◆ deinterleave_audio()

static bool LumasonicCodec::deinterleave_audio ( const float * input,
float * out0,
float * out1,
int samplesPerChannel )
static

Takes Left/Right interleaved audio and de-interleaves it into separate left and right sample buffers.

For stereo, if the total interleaved input buffer length is 512 samples, then samplesPerChannel will be 256 => 512 (input buffer length) / 2 (channels) = 256 samples.

Parameters
inputThe left/right interleaved stereo audio samples.
out0The pointer to the buffer that will contained the de-interleaved left samples.
out1The pointer to the buffer that will contained the de-interleaved right samples.
samplesPerChannelThe total number of de-interleaved audio samples per channel (for stereo this is half of the total interleaved samples in the input).
Returns
True if the process was successful, False it not.
