◆ LumasonicThreadModes

enum class LumasonicThreadModes
strong

Different threading modes for getting data from the audio thread to a worker thread.

Enumerator
LS_Thread_Sleep 

The thread will sleep by ~1 millisecond every loop while it waits for data; system sleep timing can introduce a small amount of jitter.

LS_Thread_Event 

The thread waits to be notified via a conditional event (good performance but not strictly realtime audio safe)

LS_Thread_Loop 

An aggressive thread that loops without stopping waiting for data (best timing performance, but worse CPU performance)
