Introduction to Lumasonic

Lumasonic is a high performance codec for encoding light and sound signals simultaneously into an audio signal. Much like AudioStrobe and SpectraStrobe, it utilizes specific frequency bands in the audio signal to encode control signal values that can then drive RGB LEDs.

The codec is extremely lightweight and embeddable, and intended to enable a growing ecosystem of multisensory and neural entrainment hardware.

About this SDK

This SDK provides a portable C++ codec implementation of a Lumasonic decoder and reader. With these two classes it is possible to decode and consume a stereo Lumasonic color signal from 32-bit floating point audio in realtime.

Hardware and software products can use this SDK to implement Lumasonic decoding support. This allows for integration with the Lumasonic creative tools, such as Prism Studio.
