◆ ls_codec_deinterleave_audio()

bool ls_codec_deinterleave_audio ( const float * input,
float * out0,
float * out1,
int samples_per_channel )

De-interleaves 32 bit PCM float audio, from a single interleaved stereo buffer, into two separate left and right channel buffers.

For stereo, if the total interleaved input buffer length is 512 samples, then samples_per_channel will be 256 => 512 (input buffer length) / 2 (channels) = 256 samples.

Parameters
inputThe left/right interleaved stereo audio samples.
out0The pointer to the buffer that will contained the de-interleaved left samples.
out1The pointer to the buffer that will contained the de-interleaved right samples.
samples_per_channelThe total number of de-interleaved audio samples per channel (for stereo this is half of the total interleaved samples in the input).
Returns
True if the process was successful, False it not.
