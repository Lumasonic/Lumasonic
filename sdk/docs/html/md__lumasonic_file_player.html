Lumasonic File Player

The Lumasonic File Player is a GUI tool for testing and tuning the Lumasonic Decoder. It is built with the same static library that ships with this SDK.

It can be found in the tools directory of the SDK package.

It serves several purposes:

  • testing & verifying the playback of Lumasonic encoded content on a target architecture
  • debugging decoded stereo color output values
  • tuning jitter and CPU usage by adjusting audio device drivers and reader thread mode

Audio Device Settings

Click the gear icon in the toolbar to open the Audio Settings window.

Use this window to change the audio device settings and audio driver type used for playback and decoding.

Make sure to set the sample rate to 48000 Hz and target an audio buffer size of ideally no more than 256 samples.

Read Thread Settings

Click the speedometer icon in the toolbar to open the Thread Settings window.

Use this window to change the thread mode of the reader thread.

After clicking the Apply button, observe the difference in timing performance metrics.
