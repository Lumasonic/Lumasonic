De-interleaves 32-bit PCM float audio from a single interleaved stereo buffer into four separate channel buffers 1/4th the size of the input.

Parameters
inputThe interleaved four channel audio sample buffer.
out0The pointer to channel 0's de-interleaved sample buffer.
out1The pointer to channel 1's de-interleaved sample buffer.
out2The pointer to channel 2's de-interleaved sample buffer.
out3The pointer to channel 3's de-interleaved sample buffer.
samplesPerChannelThe total number of de-interleaved audio samples per channel (for four channels this is a quarter of the total interleaved samples of the input).
Returns
True if the process was successful, False it not.

De-interleaves 32-bit PCM float audio from a single interleaved stereo buffer into two separate left and right channel buffers of half the size.

Parameters
inputThe left/right interleaved stereo audio samples.
outLThe pointer to the buffer that will contained the de-interleaved left samples.
outRThe pointer to the buffer that will contained the de-interleaved right samples.
samplesPerChannelThe total number of de-interleaved audio samples per channel (for stereo this is half of the total interleaved samples of the input).
Returns
True if the process was successful, False it not.
